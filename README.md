課程大綱分析 版本1.1
介紹
歡迎使用課程大綱分析專案！本程式旨在分析 Microsoft Word 和 PDF 格式的課程大綱文件，幫助使用者有效率地將課程大綱資訊整理成可執行的計畫。 1.1 版本新增了多項功能和改進，讓課程大綱分析和日程安排產生更加便利。

特徵
文件分析：分析 Microsoft Word 和 PDF 格式的課程大綱文件。
批次處理：同時處理多個文件，簡化分析工作流程。
個人化課程表產生：根據課程大綱資訊為學生建立個人化的每週課程表。
獨立可執行檔：透過獨立可執行檔輕鬆存取程序，實現無縫使用。
增強的使用者介面：享受更友善的使用者介面，從而改善導航和互動體驗。
要求
Python 3.12
tkinter
python-docx
組合類型
pdf2docx
用法
將程式碼倉庫複製到本機。
請確保您已安裝Python。
使用 pip 安裝所需的依賴項：
pip install tk python-docx comtypes pdf2docx
運行程式：
python main.py
點選按鈕選擇要分析的 Word 檔案。
查看產生的每週行程。
